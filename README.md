# 🎵 Ear Training

Solfege ear training in the browser. Listen, tap, build pitch memory.

**[Open app](https://harryyang21.github.io/ear-training/)** | **[Stats](https://harryyang21.github.io/ear-training/stats.html)**

---

## 🚀 Start

1. Open the link in Safari or Chrome
2. Choose Mode, Range, BPM
3. Tap Start (headphones recommended)
4. Tap keys or chords anytime to preview sounds

---

## 🎯 Modes

| Mode | What it does |
|------|----------------|
| Passive 🎧 | Two beats to listen, then auto reveal |
| Interactive 👆 | A440, one beat, tap the key |
| Bass 🎸 | Low range E1-C3, bass sound, tap to answer |
| Intervals 🎯 | Harmonic or melodic, pick interval size |
| Melody 🎼 | Short melody, replay on keyboard |
| Chords 🎹 | Triads, inversions, 7ths, progressions |

Chords mode includes 18 pop progressions (I-V-vi-IV, ii-V-I, etc.).

---

## ✨ Features

- 🎹 7 instruments: piano, violin, guitar, guzheng, erhu, harp, saxophone
- 🌈 Rainbow solfege colors for Do Re Mi Fa So La Ti
- 🧠 Adaptive practice: weak items repeat more
- ⏱️ Stats: daily goal, streak, heatmap, accuracy charts
- 📱 Mobile-friendly, add to Home Screen
- 🔁 Replay, A440, Pause, Next

---

## 📊 Stats

- Total practice time
- Accuracy by mode, note, interval, chord
- Weakest areas list

Time counts only while audio plays.

---

## 💡 Tips

- 🎧 Headphones, quiet room
- 📱 iPhone: Add to Home Screen
- 🎵 Melody mode: keyboard Z-M during replay
